# 잇쇼 (Itsho) - 일본 현지 맛집 큐레이션 서비스

> "늦은 밤, 골목 어귀. 조용히 문을 여는 한 그릇의 식당."  
> "그곳엔 여행자가 아닌, 사람이 있었다."  
> **"잇쇼 – 함께여서 따뜻한 식당들."**

잇쇼는 일본의 숨겨진 로컬 맛집을 한국인 여행자에게 소개하는 웹 기반 서비스입니다.

## 🎯 서비스 개요

### 타겟 고객
- 일본 여행을 떠나는 20~40대 한국인

### 차별화 포인트
- 일본어 정보에 의존하는 기존 타베로그와 달리 **한국어 UI + 현지 큐레이션** 제공
- 현지인이 직접 추천하는 진짜 맛집 정보

### 가치 제안
- **"현지인이 알려주는 진짜 맛집"**
- 관광지가 아닌 현지인의 일상 속 따뜻한 공간 소개

## 🌐 주요 기능

### 1. 홈페이지
- 브랜드 슬로건과 감성적 메시지 전달
- 지역 기반 맛집 검색 기능
- 인기 맛집 리스트 표시
- 서비스 특장점 안내

### 2. 맛집 리스트 페이지
- **필터링**: 지역별, 음식 카테고리별 검색
- **정렬**: 평점순, 리뷰순, 이름순
- **지도 보기**: Google Maps 연동 예정
- **북마크**: 관심 맛집 저장 기능

### 3. 맛집 상세 페이지
- 고해상도 사진 갤러리
- Google Maps 위치 정보
- 운영시간, 예약 여부, 가격대 정보
- **한국인 맞춤 후기**: 한국어로 작성된 생생한 경험담
- 교통 정보 및 접근성 안내

### 4. 커뮤니티
- **맛집 리뷰**: 개별 맛집에 대한 상세 후기
- **여행기**: 지역별 맛집 투어 경험담
- 좋아요, 댓글, 공유 기능
- 인기 주제별 게시글 분류

### 5. 마이페이지
- **저장한 맛집**: 북마크한 맛집 관리
- **내가 쓴 리뷰**: 작성한 리뷰 모아보기
- **여행 일정**: 맛집 기반 여행 계획 관리

## 🛠 기술 스택

- **Framework**: Next.js 15.5.3 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Runtime**: Node.js
- **Package Manager**: npm

## 🚀 설치 및 실행

### 개발 환경 설정

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 프로덕션 빌드
npm run build

# 프로덕션 서버 실행
npm start
```

### 개발 서버 접속
- Local: http://localhost:3000
- Network: http://[IP]:3000

## 📱 페이지 구조

```
/                    # 홈페이지 (메인 랜딩)
├── /restaurants     # 맛집 리스트
│   └── /[id]       # 맛집 상세 정보
├── /community       # 커뮤니티 (리뷰 & 여행기)
└── /mypage         # 마이페이지
```
